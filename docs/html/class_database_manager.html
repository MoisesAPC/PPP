Detailed Description

DatabaseManager singleton class definition.

This singleton handles database-related operations. This class is a wrapper for database tasks. The specific operations are handled by the currently-loaded database (see the "database" member variable).

Database* DatabaseManager::database = nullptr

database

Stores the current "Database" class instance. Depending on the database type, this will allocate a inherited class of Database, which will perform the operations specific to said database type. (for example: for CouchDB, "DatabaseCouch" will be instantiated here).
