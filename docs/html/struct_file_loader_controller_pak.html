Detailed Description

FileLoaderControllerPak class.

This class handles reading and writing .mpk / .pak files (Controller Pak containers).

The "note table" referenced below is a table that contains metadata for each save, including the start offset of the raw data associated to each save, its region, etc.

◆ getRawDataOffsetPerEntry()

unsigned int FileLoaderControllerPak::getRawDataOffsetPerEntry ( unsigned int rawDataStartOffsetByte) const
inlinevirtual

getRawDataOffsetPerEntry

Given a single raw data offset byte, it calculates the actual offset where the save can be found relative to the start of the whole binary file

For example, for Controller Paks, if the raw data byte is associated to a given save is 0x05, then the actual offset where the raw data for that save starts is at 0x05 * 0x100 = 0x500
